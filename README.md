# Week 2 Data Analysis Project

## Project Overview
Data analysis project using Excel to calculate descriptive statistics, identify trends, detect outliers, and generate insights.

## Objective
- Calculate basic statistics
- Identify trends
- Detect outliers
- Summarize findings

## Tools Used
- Microsoft Excel

## Analysis Performed
- Mean
- Median
- Count
- Min & Max
- Outlier Detection
- Trend Analysis

## Key Findings
- The average Total Price was $1,053.97.
- The median Total Price was $823.62.
- Potential outliers were identified using the IQR method.
- Most transactions were concentrated within a specific price range.
